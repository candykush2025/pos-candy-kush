## Responsive Behavior

### Mobile View (< 640px)

- Search input: Full width
- List: Full width
- Items: Larger touch targets (py-4 instead of py-3)
- Text: Slightly larger font

### Tablet View (640px - 1024px)

- Modal: 80% width
- Search: Standard size
- List: Standard height

### Desktop View (> 1024px)

- Modal: Fixed max-width
- Hover states: More prominent
- Cursor: Pointer on items

```javascript
const filteredCategories = useMemo(() => {
  if (!searchQuery.trim()) return categories;
  const query = searchQuery.toLowerCase();

}, [categories, searchQuery]);
```
